## 🚀 Запуск тестов

### Все тесты
```bash
npm run test
```

### По секциям
```bash
# Обучение
npm run test -- learning

# Тренажёр
npm run test -- trainer

# Экзамен
npm run test -- exam

# Статистика
npm run test -- statistics
```

### По группам тестов
```bash
# UI тесты
npm run test -- ui

# Math тесты
npm run test -- math

# State тесты
npm run test -- state

# Filter тесты
npm run test -- filter

# Firebase тесты
npm run test -- firebase
```

### Один файл
```bash
npm run test -- src/tests/learning/math/shuffleArray.math.test.ts --run
```

## 🐛 Известные проблемы

### UI тесты LearningProgressBar (12 тестов)

**Проблема:** Тесты используют `toHaveClass` с неправильными селекторами

**Решение:** Требуется обновить тесты на соответствие актуальным классам Tailwind

**Влияние:** Не критично, функциональность работает корректно

---

### Filter тесты Learning (4 теста)

**Проблема:** Ожидания не соответствуют реальному поведению хука

**Решение:** Обновить тесты на соответствие реализации `useLearningFilter`

**Влияние:** Не критично, фильтр работает корректно
