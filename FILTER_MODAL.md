# ✅ Фильтр вопросов в модальном окне

## Изменение
Блок с настройкой фильтров перемещён из раскрывающейся панели в модальное окно.

## Преимущества

### UX улучшения
- ✅ **Больше пространства** — фильтр не занимает место на странице
- ✅ **Фокус на настройках** — модальное окно концентрирует внимание
- ✅ **Предварительный просмотр** — показывается количество вопросов до применения
- ✅ **Анимации** — плавное появление и исчезновение
- ✅ **Закрытие по клику вне** — интуитивное управление

### Визуальные улучшения
- ✅ **4 карточки статистики** — Известные, Слабые, В работе, Всего
- ✅ **Живой предпросмотр** — количество вопросов обновляется при изменении настроек
- ✅ **Кнопка с количеством** — "Применить (150 вопр.)"
- ✅ **Скролл для списка** — компактное отображение вопросов

## Как использовать

### Открытие фильтра
```typescript
// Кнопка в LearningSection
<Button onClick={() => setIsFilterModalOpen(true)}>
  <Filter className="w-4 h-4" />
  <span>Фильтр</span>
</Button>
```

### Закрытие фильтра
```typescript
// Автоматически после применения
onApply={(filteredIds, settings) => {
  // Применяем фильтр
  setIsFilterModalOpen(false); // Закрываем модальное окно
}}

// Или по кнопке X в заголовке
<Button onClick={onClose}>
  <X className="w-4 h-4" />

```

## Компоненты модального окна

### Карточки статистики
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│  Известные  │   Слабые    │   В работе  │    Всего    │
│     50      │     30      │    170      │     250     │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

### Переключатели
- Исключить известные (100% точность)
- Исключить слабые (< 70% точность)

### Ручное скрытие
- Список вопросов (20 на странице)
- Кнопка "глаз" для скрытия/показа
- Прокрутка если вопросов > 20

### Кнопки действий
- **Сбросить фильтры** — сбрасывает все настройки
- **Применить (N вопр.)** — применяет фильтр и закрывает окно
