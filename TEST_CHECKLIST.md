### 1. Фильтр вопросов

#### TC-1.1: Применение фильтра
```
Шаги:
1. Открыть раздел "Обучение"
2. Нажать кнопку "Фильтр"
3. Развернуть панель (если свёрнута)
4. Включить "Исключить известные"
5. Нажать "Применить"

Ожидаемый результат:
✅ Вопросы перестроились (исключены известные)
✅ Прогресс-бар обновился (показывает меньшее количество)
✅ Глобальный прогресс пересчитан
✅ В консоли: "🔍 [LearningSection] Фильтр применён, вопросов: X"
```

#### TC-1.2: Сброс фильтра
```
Шаги:
1. Применить фильтр (TC-1.1)
2. Нажать "Сбросить фильтры"

Ожидаемый результат:
✅ Все вопросы вернулись
✅ Прогресс-бар показывает полное количество
✅ В консоли: "🔍 [LearningSection] Фильтр применён, вопросов: X"
```

#### TC-1.3: Ручное скрытие вопросов
```
Шаги:

3. Нажать "Применить"

Ожидаемый результат:
✅ Скрытый вопрос исключён из списка
✅ В фильтре показано "Скрыто вопросов: 1"
```
