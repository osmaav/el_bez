## 🛠️ Новые скрипты

Добавлены утилиты для управления базой данных Firestore:

| Команда | Назначение |
|---------|------------|
| `npm run db:backup` | Резервное копирование Firestore |
| `npm run db:restore` | Восстановление из бэкапа |
| `npm run db:migrate` | Пошаговая миграция с подтверждением |
| `npm run db:migrate:auto` | Автоматическая миграция без пауз |
| `npm run db:check` | Проверка состояния базы |
| `npm run db:verify [section]` | Детальная проверка секции |
| `npm run db:fix:correct` | Исправление поля `correct` |
| `npm run db:fix:sessions` | Исправление сессий пользователей |
