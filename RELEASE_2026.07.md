# Релиз 2026.07 — Экспорт результатов в PDF

**Дата релиза:** 7 марта 2026 г.  
**Версия:** 2026.07.0

## 🎯 Основные изменения

### ✨ Экспорт результатов в PDF

Добавлена возможность экспорта результатов в PDF во всех режимах обучения:

#### 📚 Обучение
- Экспорт текущей страницы с вопросами и ответами
- Статистика страницы (правильно/неправильно/осталось)
- Глобальный прогресс по разделу
- Детальный разбор с вашими ответами и правильными вариантами
- Имя файла: `el-bez_learning_page_{page}_{timestamp}.pdf`

#### 🏋️ Тренажёр
- Экспорт результатов тренировки (20 или 50 вопросов)
- Общая статистика с процентом правильных ответов
- Детальный разбор всех вопросов с указанием верных ответов
- Имя файла: `el-bez_trainer_{timestamp}.pdf`

#### 📝 Экзамен
- Экспорт результатов экзамена по билету
- Результат сдачи (сдал/не сдал) с цветовой индикацией
- Статистика по билету (80% для успешной сдачи)
- Детальный разбор всех 10 вопросов билета
- Имя файла: `el-bez_exam_ticket_{id}_{passed|failed}_{timestamp}.pdf`

### 🔧 Технические улучшения

- **Шрифт Roboto Regular** — встроенный шрифт с поддержкой кириллицы
- **jsPDF + autoTable v5** — генерация PDF с таблицами
- **Автоматическая загрузка шрифта** — из CDN с кэшированием
- **Корректная кодировка** — поддержка UTF-8 и кириллических символов
- **Цветовая схема** — согласованная с дизайном приложения

## 🚀 Использование

### Обучение
1. Пройдите вопросы на странице
2. После ответа на все вопросы нажмите «Сохранить в PDF»
3. Файл сохранится в папку загрузок

### Тренажёр
1. Завершите тренировку (20 или 50 вопросов)
2. На экране результатов нажмите «Сохранить в PDF»
3. Файл сохранится в папку загрузок

### Экзамен
1. Завершите экзамен (ответьте на все 10 вопросов билета)
2. На экране результатов нажмите «Сохранить в PDF»
3. Файл сохранится в папку загрузок

## 🎨 Формат PDF

Все PDF-файлы содержат:
- **Шапку** — название раздела, дата генерации
- **Статистику** — визуальное отображение результатов
- **Таблицу** — вопросы, ваши ответы, правильные ответы, статус
- **Подвал** — информация о приложении

## ⚠️ Известные ограничения

- При первом экспорте возможна задержка 1-2 секунды на загрузку шрифта (~164KB)
- Последующие экспорты работают быстрее (шрифт кэшируется в памяти)
- Для работы требуется подключение к интернету (загрузка шрифта из CDN)
