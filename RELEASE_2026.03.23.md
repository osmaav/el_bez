## 🛠 ETL-инструменты

Созданы скрипты для работы с базой данных вопросов:

### 1. `scripts/backup-firestore-questions.cjs`
Резервное копирование всех вопросов из Firestore в JSON.

**Использование:**
```bash
node scripts/backup-firestore-questions.cjs
```

**Результат:** `scripts/backups/questions-backup-YYYY-MM-DD_HH-MM-ss.json`

### 2. `scripts/analyze-and-compare.cjs`
Анализ и сравнение вопросов в Firestore с файлами.

**Использование:**
```bash
node scripts/analyze-and-compare.cjs
```

**Результат:**
- Отчёт о неполных билетах
- Список отсутствующих вопросов
- Статистика по разделам

### 3. `scripts/upload-questions-full.cjs`
Загрузка вопросов из `scripts/questions-full/` в Firestore.

**Использование:**
```bash
node scripts/upload-questions-full.cjs
```

**Особенности:**
- Интерактивное подтверждение
- Дополнение неполных билетов
- Обновление изменённых вопросов
- Пакетная загрузка по 500 вопросов
