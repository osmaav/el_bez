# Релиз 2026.05 — Улучшения UX и стабильности

**Дата релиза:** 1 марта 2026 г.

**Версия:** 2026.05.0

---

## 🎉 Основные нововведения

### 🔐 Модальные окна входа и регистрации

Полный отказ от роутинга в пользу модальных окон:

- **Удалены страницы** `/login` и `/register`
- **Модальное окно входа** — открывается из навигационной панели
- **Модальное окно регистрации** — переключение из окна входа
- **Кнопка "Войти"** в навигации — видна на всех экранах
- **Информационный текст** — описание назначения ресурса в окне входа

### 👁️ Просмотр пароля

Удобное управление видимостью пароля:

- **Кнопка с иконкой глаза** в полях ввода пароля
- **Переключение видимости** — клик по иконке
- **Иконка Eye/EyeOff** — визуальная индикация состояния
- **Реализовано** в LoginModal и RegisterModal

### 💾 Раздельное хранение прогресса

Полная изоляция прогресса для разделов:

- **electrospa_learning_page_1256-19** → `elbez_learning_page_1256-19`
- **electrospa_learning_page_1258-20** → `elbez_learning_page_1258-20`
- **electrospa_learning_progress_*** → `elbez_learning_progress_*`
- **Валидация страницы** — автоматический сброс при недоступности
- **Кнопка "Сброс"** — очищает прогресс только текущего раздела

### 📱 Адаптивная навигация

Оптимизация для различных размеров экрана:

- **< 930px** — только иконки в навигации
- **≥ 930px** — иконки + подписи
- **Кнопка "Войти"** — иконка всегда видна, текст при ≥ 930px
- **Статичный заголовок** — "Обучение" вместо динамического

### ⚡ Кэширование вопросов

Ускорение переключения между разделами:

- **questionsCache (Map)** — хранение загруженных вопросов
- **Проверка кэша** — перед загрузкой из Firestore
- **Мгновенная загрузка** — при повторном переключении
- **Логирование** — кэш/Firestore загрузка

### 🔍 Расширенное логирование

Полная отладка работы приложения:

- **authService.ts** — логирование авторизации
- **questionService.ts** — загрузка вопросов
- **AuthContext.tsx** — состояние аутентификации
- **AppContext.tsx** — загрузка вопросов, кэширование
- **LoginModal.tsx** — процесс входа
- **RegisterModal.tsx** — процесс регистрации
- **LearningSection.tsx** — сохранение/загрузка прогресса

### 🚫 Запоминание отказа от входа

Уважение выбора пользователя:

- **elbez_declined_login** — флаг отказа в localStorage
- **Не показывать окно** — при повторной загрузке
- **Кнопка "Войти"** — для повторного открытия окна

## 🐛 Исправления

### Критические

- 🐛 **Перезапись elbez_learning_page_** — исправлено сохранение при переключении разделов
- 🐛 **Восстановление страницы** — добавлено подробное логирование saveCurrentPage/loadCurrentPage

- 🐛 **Заголовок "Обучение"** — статический вместо динамического

## 🚀 Миграция на версию 2026.05

### Для пользователей

**Важно:** При обновлении ключи localStorage будут автоматически переименованы с `electrospa_` на `elbez_`.

1. **Очистите кэш браузера** (опционально)
2. **Обновите страницу** (F5)
3. **Войдите в систему** (если требуется)

### Для разработчиков

```bash
# Обновление зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```
