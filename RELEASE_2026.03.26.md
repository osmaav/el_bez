## 🎯 Как это работает

### Запуск экзамена
```tsx
// При старте экзамена таймер автоматически запускается
handleStartExam(ticketId) {
  startExam(ticketId);
  resetTimer();  // Сброс к 30 минутам
  startTimer();  // Запуск отсчёта
}
```

### Истечение времени
```tsx
// Когда время истекает (0 секунд)
useExamTimer({
  onStopListening: () => {
    // Автоматическое завершение экзамена
    if (!isExamFinished) {
      finishExam();
    }
  }
})
```

### UI компонент
```tsx
<ExamTimer
  timeLeft={timeLeft}           // Оставшееся время (сек)
  formattedTime={formattedTime} // "MM:SS"
  isActive={isTimerActive}      // Активен ли таймер
  isFinished={isTimerFinished}  // Истекло ли время
  onStart={startTimer}          // Кнопка "Старт"
  onStop={stopTimer}            // Кнопка "Пауза"
  onReset={resetTimer}          // Кнопка "Сброс"
/>
```

## 📝 Примеры использования

### В режиме экзамена

1. **Пользователь начинает экзамен:**
   - Таймер показывает `30:00`
   - Кнопка "Старт" активна

2. **Пользователь нажимает "Старт":**
   - Таймер начинает отсчёт: `29:59`, `29:58`, ...
   - Кнопка меняется на "Пауза"

3. **Осталось 5 минут:**
   - Таймер становится жёлтым
   - Текст: "Осталось 5 минут"

4. **Осталась 1 минута:**
   - Таймер становится оранжевым
   - Текст: "Осталась 1 минута"

5. **Осталось 30 секунд:**
   - Таймер становится красным
   - Текст: "Осталось 30 секунд!"

6. **Время вышло:**
   - Таймер показывает `00:00`
   - Экзамен автоматически завершается
   - Показывается сообщение "Время вышло!"

## 🔧 API хука useExamTimer

### Параметры
```typescript
interface UseExamTimerOptions {
  onStopListening?: () => void;  // Колбэк при истечении времени
}
```

### Возвращаемое значение
```typescript
interface UseExamTimerReturn {
  timeLeft: number;        // Оставшееся время в секундах
  formattedTime: string;   // Форматированное время "MM:SS"
  isActive: boolean;       // Активен ли таймер
  isFinished: boolean;     // Истекло ли время
  start: () => void;       // Запустить таймер
  stop: () => void;        // Остановить таймер
  reset: () => void;       // Сбросить к 30 минутам
}
```

### Пример использования
```typescript
const {
  timeLeft,        // 1800 → 0
  formattedTime,   // "30:00" → "00:00"
  isActive,        // true/false
  isFinished,      // true/false
  start,           // Запуск
  stop,            // Пауза
  reset,           // Сброс
} = useExamTimer({
  onStopListening: () => {
    console.log('Время вышло!');
  }
});
```
