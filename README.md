# ЭБ 1256.19 / ЭБ 1258.20 — Тесты Ростехнадзора по электробезопасности

**el_bez** — современное веб-приложение для подготовки к сдаче экзаменов по электробезопасности (III и IV группа до 1000 В).

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-2026.05-green.svg)](CHANGELOG.md)

## 📖 Описание

Приложение предназначено для самостоятельной подготовки к проверке знаний по электробезопасности. Содержит актуальные вопросы из тестов Ростехнадзора для III и IV группы допуска до 1000 В.

### Особенности

- ✅ **560 вопросов** из актуальной базы Ростехнадзора
- ✅ **Два раздела:** ЭБ 1256.19 (III группа) и ЭБ 1258.20 (IV группа)
- ✅ **56 экзаменационных билетов** по 10 вопросов
- ✅ **Firebase Firestore** — облачное хранение вопросов и состояний пользователей
- ✅ **Система аутентификации** — модальные окна входа и регистрации
- ✅ **Просмотр пароля** — кнопка с иконкой глаза в полях ввода
- ✅ **Режим обучения** — вопросы разбиты на страницы по 10 вопросов
- ✅ **Тренажёр** — случайная выборка вопросов с мгновенной проверкой ответа
- ✅ **Экзамен** — имитация реального тестирования
- ✅ **Раздельное хранение прогресса** — для каждого раздела отдельно
- ✅ **Кэширование вопросов** — мгновенное переключение между разделами
- ✅ **Автозаполнение форм** — данные форм сохраняются для быстрого входа
- ✅ **Адаптивный дизайн** — оптимизирован для экранов < 930px и ≥ 930px

## 🛠 Технологии

- **Frontend:** React 19, TypeScript, Vite
- **UI:** Tailwind CSS, shadcn/ui, Radix UI
- **State Management:** React Context API
- **Backend:** Firebase Firestore (вопросы, состояния пользователей)
- **Auth:** Firebase Authentication (email/password, OAuth)
- **Storage:** 
  - Firestore — хранение вопросов и состояний пользователей
  - localStorage — кеширование, автозаполнение, настройки
- **Build:** npm, Vite
- **Migration:** firebase-admin (скрипт переноса данных)

## 📦 Установка

### Требования

- Node.js 20+
- npm 10+

### Локальный запуск

```bash
# Клонирование репозитория
git clone https://github.com/osmaav/el_bez.git
cd el_bez/app

# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```

### Настройка Firebase

Для работы с облачной базой данных:

1. Создайте проект в [Firebase Console](https://console.firebase.google.com/)
2. Включите Authentication (Email/Password)
3. Создайте Firestore Database
4. Скопируйте конфигурацию в `.env.local`:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

5. Загрузите вопросы в Firestore:

```bash
# Установите firebase-admin
npm install firebase-admin

# Запустите миграцию
node scripts/migrate-questions-to-firestore.cjs
```

Подробная инструкция в [FIRESTORE_MIGRATION.md](FIRESTORE_MIGRATION.md).

### Развёртывание на GitHub Pages

Проект настроен для автоматического деплоя через GitHub Actions:

1. Включите GitHub Pages в настройках репозитория
2. Выберите источник: GitHub Actions
3. При пуше в ветку `master` сборка и деплой выполняются автоматически

## 🎯 Использование

### Переключение между разделами

1. Нажмите на название раздела в верхней панели (например, "ЭБ 1258.20")
2. Выберите нужный раздел из выпадающего списка:
   - **ЭБ 1256.19** — III группа до 1000 В (250 вопросов, 25 билетов)
   - **ЭБ 1258.20** — IV группа до 1000 В (309 вопросов, 31 билет)

### Режим обучения

1. Перейдите в раздел **«Обучение»**
2. Отвечайте на вопросы (по 10 на странице)
3. Используйте кнопки **«Далее»** для перехода между страницами
4. После ответа на все вопросы страницы нажмите **«Далее...»**
5. На последней странице нажмите **«Новая сессия»** для сброса

### Тренажёр

1. Выберите **«Тренажёр»** в меню
2. Нажмите **«Начать тренировку (50 вопросов)»** или **«Короткая тренировка (20 вопросов)»**
3. Отвечайте на вопросы, получая мгновенную обратную связь
4. Используйте навигацию для возврата к предыдущим вопросам
5. После завершения просмотрите детальную статистику

### Экзамен

1. Выберите **«Экзамен»** в меню
2. Выберите билет (1–25 для 3 группы, 1–31 для 4 группы)
3. Ответьте на все 10 вопросов
4. Нажмите **«Завершить»** для получения результатов
5. Для успешной сдачи необходимо ответить правильно на 80% вопросов (8 из 10)
