# 📊 El-Bez Release 2026.03.29.00

## 🎯 Основные изменения

### Улучшения страницы "Статистика"

#### 1. Динамическое отображение разделов в "Прогресс по разделам"
- ✅ Показываются **только разделы, по которым есть результаты**
- ✅ **Поддержка всех 19 разделов** (промышленные, непромышленные, ЭТЛ)
- ✅ Сообщение "Пока нет результатов" при отсутствии данных

#### 2. Полное описание группы в "Темы для повторения"
- ✅ **Бейдж показывает полное описание раздела** из констант
- ✅ **Поддержка всех 19 разделов**
- ✅ Примеры: "III группа до 1000 В", "IV группа до 1000 В (непромышленные)", "V группа до и выше 1000 В (ЭТЛ)"

#### 3. Динамический подсчёт времени обучения
- ✅ Общее время обучения считается **по всем разделам**
- ✅ Удалены жёсткие ссылки на конкретные разделы
