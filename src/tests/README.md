# 📚 Инструкция по тестированию el-bez

## Структура тестов

```
src/tests/
├── learning/          # Тесты секции "Обучение"
│   ├── ui/           # UI тесты
│   ├── math/         # Тесты вычислений
│   ├── state/        # Тесты состояний
│   ├── filter/       # Тесты фильтра
│   └── firebase/     # Firebase тесты
├── trainer/          # Тесты секции "Тренажёр"
├── exam/             # Тесты секции "Экзамен"
└── statistics/       # Тесты секции "Статистика"
```

---

## 🚀 Запуск тестов

### Все тесты
```bash
npm run test
```

### Тесты по секции
```bash
# Обучение
npm run test -- learning

# Тренажёр
npm run test -- trainer

# Экзамен
npm run test -- exam

# Статистика
npm run test -- statistics
```

### Тесты по группе
```bash
# UI тесты
npm run test -- ui

# Математика
npm run test -- math

# Состояния
npm run test -- state

# Фильтры
npm run test -- filter

# Firebase
npm run test -- firebase
```

### Один файл
```bash
npm run test -- LearningSection.ui.test.ts
```

## 📋 Секция: Обучение (Learning)

### UI Тесты

**Что тестируется:**
- Отображение заголовка
- Прогресс-бар
- Кнопки навигации
- Карточки вопросов
- Варианты ответов

**Запуск:**
```bash
npm run test -- learning/ui
```

**Пошаговая инструкция:**
1. Откройте терминал
2. Запустите: `npm run test -- learning/ui`
3. Проверьте, что все тесты зелёные ✅
4. При ошибке изучите лог

---

### Math Тесты

**Что тестируется:**
- `shuffleArray` — перемешивание вариантов
- `calculateProgress` — расчёт прогресса
- `calculateStats` — подсчёт статистики
- `getAnswerStyle` — стили ответов

**Запуск:**
```bash
npm run test -- learning/math
```

**Пошаговая инструкция:**
1. Запустите тесты
2. Проверьте перемешивание (разные порядки)
3. Проверьте проценты (0%, 50%, 100%)
4. Проверьте стили (зелёный/оранжевый/серый)

---

### State Тесты

**Что тестируется:**
- Ответ на вопрос
- Переход между страницами
- Переход между секциями
- Перезагрузка страницы
- Восстановление прогресса

**Запуск:**
```bash
npm run test -- learning/state
```

**Пошаговая инструкция:**
1. Запустите тесты
2. Проверьте сохранение ответов
3. Проверьте переход на стр. 2 и возврат
4. Проверьте localStorage
5. Проверьте сброс прогресса

---

### Filter Тесты

**Что тестируется:**
- Включение фильтра
- Выключение фильтра
- Сброс фильтра
- Скрытые вопросы

**Запуск:**
```bash
npm run test -- learning/filter
```

---

### Firebase Тесты

**Что тестируется:**
- Регистрация
- Вход
- Изменение профиля
- Удаление профиля
- Подтверждение email
- Запись в БД
- Чтение из БД

**Запуск:**
```bash
npm run test -- learning/firebase
```

## 🐛 Отладка

### Тест не проходит
1. Проверьте лог ошибки
2. Запустите с `--reporter=verbose`
3. Добавьте `console.log` в тест

### Ложное срабатывание
1. Проверьте моки
2. Обновите зависимости
3. Очистите кэш: `rm -rf node_modules/.vite`
