## 🚀 Быстрый старт

### 1. Запуск приложения

```bash

cd app
```

- **ЭБ 1258.20** — IV группа до 1000 В (310 вопросов)

## 📚 Режимы работы

### 1. Обучение 🎓

**Назначение:** Пошаговое изучение вопросов по 10 на странице с сохранением прогресса.

**Как использовать:**
1. Откройте вкладку "Обучение"
2. Выберите ответ кликом на вариант
3. Для вопросов с несколькими ответами:
   - Кликните на первый ответ → он окрасится **синим**
   - Кликните на второй ответ → он тоже окрасится **синим**
   - Когда выбраны все ответы → автоматически проверится правильность
4. При переключении на другую вкладку прогресс **сохраняется**

**Особенности:**
- ✅ Прогресс сохраняется автоматически
- ✅ Можно вернуться к предыдущим страницам
- ✅ Фильтр вопросов для эффективной подготовки
- ✅ Экспорт результатов в PDF

---

### 2. Тренажёр 🏋️

**Назначение:** Тренажёр со случайной выборкой вопросов (20 или 50).

**Как использовать:**
1. Откройте вкладку "Тренажёр"
2. Выберите количество вопросов:
   - **50 вопросов** — полная тренировка
   - **20 вопросов** — короткая тренировка
3. Для вопросов с несколькими ответами:
   - Кликните на первый ответ → он окрасится **синим** 🟦
   - Кликните на второй ответ → он тоже окрасится **синим** 🟦
   - Когда выбраны все ответы → автоматически проверится правильность
4. После ответа нажмите "Далее" или "Завершить"

**Особенности:**
- ✅ Случайная выборка вопросов
- ✅ Мгновенная обратная связь
- ✅ Статистика в реальном времени
- ✅ Экспорт результатов в PDF

**Кнопка "Автоответ":**
- Расположена в правом нижнем углу
- Автоматически отвечает на все вопросы
- Проходит по вопросам с задержкой 500мс
- Полезна для быстрого тестирования

---

### 3. Экзамен 📝

**Назначение:** Имитация реального экзамена (билеты по 10 вопросов).

**Как использовать:**
1. Откройте вкладку "Экзамен"
2. Выберите билет (1-25 для III группы, 1-31 для IV группы)
3. Отвечайте на вопросы:
   - Для одиночного выбора: кликните на ответ → сразу проверяется
   - Для множественного выбора: выберите все ответы → проверится после последнего
4. Нажмите "Завершить экзамен"

**Проходной балл:** 80% (8 из 10 правильных ответов)

**Особенности:**
- ✅ 56 экзаменационных билетов
- ✅ Детальный разбор результатов
- ✅ Экспорт билета в PDF
- ✅ Кнопка "Автоответ" для тестирования

---

### 4. Статистика 📊

**Назначение:** Просмотр прогресса и активности.

**Вкладки:**
- **Обзор** — общая статистика по всем разделам
- **ЭБ 1256.19** — статистика по III группе
- **ЭБ 1258.20** — статистика по IV группе

**Метрики:**
- Всего сессий
- Вопросов отвечено
- Точность (%)
- Время обучения
- Активность по дням
- Слабые темы

## 🎨 Интерфейс

### Цветовая схема ответов:

| Цвет | Значение |
|------|----------|
| 🔵 **Синий** | Ответ выбран (ожидание проверки) |
| 🟢 **Зелёный** | Правильный ответ |
| 🔴 **Красный** | Неправильный ответ |
| ⚪ **Серый** | Ответ не выбран |

### Иконки:

| Иконка | Значение |
|--------|----------|
| ✅ | Правильно |
| ❌ | Неправильно |
| 🎯 | Автоответ |
| 📊 | Статистика |
| 📥 | Экспорт в PDF |

## 🔧 Настройки

### Фильтр вопросов

**Где:** Обучение, Тренажёр (кнопка "Фильтр")

**Опции:**
- **Скрыть вопросы** — исключить конкретные вопросы
- **Исключить известные** — скрыть вопросы со 100% точностью
- **Исключить слабые** — скрыть вопросы с точностью < 70%

**Применение:**
1. Нажмите "Фильтр"
2. Выберите настройки
3. Нажмите "Применить"
4. Вопросы отфильтруются автоматически

## 📥 Экспорт в PDF

### Где доступен:
- ✅ Обучение (после завершения страницы)
- ✅ Тренажёр (после завершения тренировки)
- ✅ Экзамен (после завершения экзамена)
- ✅ Статистика (кнопка "Экспорт")

### Что экспортируется:
- Результаты тестирования
- Детальный разбор вопросов
- Правильные и неправильные ответы
- Дата и время прохождения

**Формат имени файла:**
```
el-bez_[режим]_[дата]_[время].pdf
```

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **→** (стрелка вправо) | Следующий вопрос |
| **←** (стрелка влево) | Предыдущий вопрос |

## 🐛 Известные ограничения

1. **Множественный выбор:**
   - Для вопросов с несколькими ответами необходимо выбрать **все** варианты
   - Нельзя выбрать больше ответов, чем требуется
   - Можно снять выделение повторным кликом

2. **Автоответ:**
   - Работает только в Тренажёре и Экзамене
   - Отвечает случайными ответами
   - Не предназначен для обучения

3. **Экспорт в PDF:**
   - Длинные ответы (>50 символов) усекаются
   - Кириллические символы требуют загрузки шрифта

## 💡 Советы по использованию

### Для эффективной подготовки:

1. **Начните с Обучения:**
   - Пройдите все вопросы по 10 на странице
   - Используйте фильтр для исключения известных тем
   - Сохраняйте прогресс

2. **Перейдите к Тренажёру:**
   - Выберите 50 вопросов для полной тренировки
   - Используйте автоответ для быстрого тестирования
   - Экспортируйте результаты для анализа

3. **Сдайте Экзамен:**
   - Выберите случайный билет
   - Постарайтесь набрать ≥80%
   - Изучите детальный разбор ошибок

4. **Анализируйте Статистику:**
   - Проверьте слабые темы
   - Отследите прогресс по дням
   - Скорректируйте план подготовки
